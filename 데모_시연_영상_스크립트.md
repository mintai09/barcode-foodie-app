# 데모 시연 영상 스크립트 (3-5분)

---

## 🎬 영상 구성

**총 길이**: 약 4분
**화면**: 실제 앱 화면 캡처 + 내레이션
**배경음악**: 밝고 따뜻한 분위기

---

## 장면 1: 오프닝 (15초)

**화면**: 타이틀 화면
```
🍔 들리는 바코드 푸디
Barcode Foodie

시각장애인과 알레르기 환자를 위한
AI 맞춤 식료품 비서 서비스
```

**내레이션**:
> "안녕하세요. 들리는 바코드 푸디를 소개합니다.
> 시각장애인과 알레르기 환자가 안전하게 식품을 선택할 수 있도록
> AI 기술로 만든 웹 애플리케이션입니다."

---

## 장면 2: 문제 정의 (20초)

**화면**: 문제 상황 텍스트 + 이미지
```
❓ 기존의 문제점

1. 시각장애인
   - 제품 정보 확인 불가능
   - 알레르기 성분 파악 어려움

2. 알레르기 환자
   - 복잡한 성분표 이해 어려움
   - 숨겨진 알레르겐 탐지 불가
```

**내레이션**:
> "시각장애인은 제품의 바코드를 스캔하기 어렵고,
> 알레르기 환자는 성분표에 숨어있는 위험 성분을 찾기 힘듭니다.
> 이 문제를 AI 기술로 해결했습니다."

---

## 장면 3: AI 기술 소개 (30초)

**화면**: AI 기술 4가지 아이콘 + 간단 설명
```
🤖 적용된 AI 기술

1️⃣ Computer Vision
   바코드 패턴 자동 감지 (90% 정확도)

2️⃣ NLP 챗봇
   Gemma-3-12B-IT 의료 문진

3️⃣ Speech AI
   STT/TTS 음성 인터페이스

4️⃣ 위험도 분석
   교차반응 및 심각도 판정
```

**내레이션**:
> "4가지 AI 기술을 적용했습니다.
> 첫째, Computer Vision으로 이미지에서 바코드를 자동으로 찾고,
> 둘째, NLP 챗봇으로 알레르기 정보를 수집하며,
> 셋째, 음성 AI로 시각장애인을 지원하고,
> 넷째, 위험도 분석으로 안전성을 판정합니다."

---

## 장면 4: 실제 사용 데모 - Part 1 (알레르기 프로필 설정) (30초)

**화면**: 앱 실행 → 프로필 설정 화면

**내레이션**:
> "먼저 알레르기 프로필을 설정합니다."

**화면 동작**:
1. 앱 접속 → "바코드 스캔 화면입니다" (TTS 재생)
2. 알레르기 항목 선택: "땅콩", "우유" 클릭
3. "프로필 저장하고 계속하기" 버튼 클릭

**내레이션**:
> "음성 안내를 들으며 알레르기 항목을 선택합니다.
> 모든 과정이 음성으로 안내되어 시각장애인도 쉽게 사용할 수 있습니다."

---

## 장면 5: 실제 사용 데모 - Part 2 (AI 챗봇 문진) (45초)

**화면**: AI 챗봇 화면

**내레이션**:
> "다음은 AI 챗봇이 상세 정보를 수집합니다."

**화면 동작**:
1. 챗봇: "땅콩 알레르기의 주요 증상은 무엇인가요?"
2. 사용자: 🎙️ 마이크 버튼 클릭 → "호흡곤란이요" (음성 입력)
3. 챗봇: "심각도는 어떠신가요?"
4. 사용자: 빠른 응답 버튼 "심각함" 클릭
5. 챗봇: "아나필락시스 경험이 있으신가요?"
6. 사용자: "예" 클릭
7. 챗봇: "교차 반응 확인 중..." → "견과류도 반응하시나요?"
8. 사용자: "예" 클릭

**내레이션**:
> "AI 챗봇이 5단계로 체계적인 문진을 진행합니다.
> 음성으로 답변하거나 버튼을 눌러 간편하게 입력할 수 있습니다.
> 증상, 심각도, 아나필락시스 이력, 교차반응까지 자동으로 수집합니다."

---

## 장면 6: 실제 사용 데모 - Part 3 (바코드 스캔) (40초)

**화면**: 바코드 스캔 화면

**내레이션**:
> "이제 제품을 스캔해보겠습니다."

**화면 동작**:
1. "📁 이미지에서 바코드 찾기" 버튼 클릭
2. 파일 선택: "바코드3.jpg" 업로드
3. 로딩 화면: "이미지를 분석하고 있습니다..." (TTS)
4. 콘솔 로그 오버레이:
   ```
   🔍 바코드 패턴 감지 시작...
   ✅ 바코드 영역 감지됨: x=120, y=450, w=680, h=120
   📊 전환 횟수: 87
   ✂️ 총 14개 영역 생성됨
   🔎 영역 1/14: 🎯 감지된 바코드 영역 (최우선) 스캔 중...
   ✅ 성공! 감지된 바코드 영역에서 바코드 발견: 8801019606557
   ```
5. "바코드 8801019606557을 인식했습니다" (TTS)

**내레이션**:
> "이미지를 업로드하면 AI가 자동으로 바코드 영역을 찾습니다.
> 흰 배경과 검은 수직선 패턴을 분석하여,
> 이미지 어디에 있든 바코드를 정확히 탐지합니다.
> 14개 영역으로 나눠 순차적으로 스캔하여 90% 이상의 정확도를 달성했습니다."

---

## 장면 7: 실제 사용 데모 - Part 4 (위험도 분석 결과) (40초)

**화면**: 제품 분석 결과 화면

**화면 동작**:
1. API 호출 중 로딩
2. 결과 화면 표시:

```
🚫 섭취 금지 (위험)

제품명: 새우깡
브랜드: 농심

⚠️ 발견된 알레르기 성분:
• 새우 (위험)
• 밀 (주의)
• 대두 (주의)

⚠️ 경고:
이 제품에는 새우, 밀, 대두가 포함되어 있습니다.
같은 시설에서 계란, 우유, 게를 사용한 제품을 제조합니다.

❌ 귀하의 알레르기 프로필과 일치하는 성분이 발견되었습니다.
섭취하지 마세요.
```

3. TTS 재생: "섭취 금지. 이 제품에는 새우가 포함되어 있습니다..."
4. 버튼 클릭: "🔊 다시 듣기", "📋 원재료 듣기"

**내레이션**:
> "분석 결과가 나왔습니다.
> 사용자 프로필과 비교하여 위험 알레르겐이 발견되면 빨간색으로 표시됩니다.
> 같은 시설에서 제조된 제품의 교차오염 위험까지 알려줍니다.
> 모든 정보는 음성으로도 안내되어 시각장애인도 완벽하게 이해할 수 있습니다."

> "안전한 제품은 초록색으로 표시됩니다.
> 간단하고 직관적인 3색 코딩으로 누구나 쉽게 이해할 수 있습니다."

## 🎥 촬영 가이드

### 화면 녹화 설정
- **해상도**: 1920x1080 (Full HD)
- **프레임**: 30fps
- **음성**: 명확한 발음, 적절한 속도
- **자막**: 한글 자막 추가 (청각장애인 배려)

### 편집 포인트
1. **트랜지션**: 부드러운 페이드 효과
2. **강조**: 중요 텍스트 확대/색상 변경
3. **효과음**: 버튼 클릭, TTS 재생 시 적절한 사운드
4. **배경음악**: 볼륨 -20dB (내레이션 방해 금지)

### 추가 삽입 장면 (선택)
- 실제 사용자 인터뷰 (시각장애인/알레르기 환자)
- 슈퍼마켓에서 실제 사용 시연
- 개발 과정 비하인드 스토리
